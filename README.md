## 오케스트라 편곡 
출판을 할게 아니라면, 결국 다양한 곳에서 편곡된 찬송이 불려지기를 바랄 것입니다. 아마추어 작곡가, 아마추어 음악가들이 시도한 
편곡 작업물을 모을 수 있는 공간으로 만들 생각입니다. 

아직 종이에다가 작업하는 법이라던가, 컴퓨터 사용해서 바로 작업하는 환경이 없어서, 시도조차 못하고 있습니다.
일단 머리속에서, 음악이 들리지는 않아서.. 무조건 환경이 구성되어야 합니다.
